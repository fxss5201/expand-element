# expand-element

[文档](https://fxss5201.github.io/expand-element/)

拓展浏览器中DOM、BOM的一些方法，以方便开发。

## 安装

::: code-group

```bash [npm]
npm install expand-element
```

```bash [yarn]
yarn add expand-element
```

```bash [pnpm]
pnpm add expand-element
```

:::

## 使用

```js
import { getElementPadding } from 'expand-element'
```
